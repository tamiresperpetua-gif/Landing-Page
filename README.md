Projeto desenvolvido para um cliente que atua como corretor de imóveis e afiliado. O objetivo foi criar uma landing page de apresentação capaz de despertar o 
interesse dos visitantes pelos serviços oferecidos, conduzindo-os de forma estratégica para o contato via WhatsApp, garantindo um atendimento rápido e eficiente.

Além da conversão, a proposta da página foi priorizar uma experiência leve, intuitiva e visualmente agradável, transmitindo credibilidade, confiança e profissionalismo 
sem abrir mão da simplicidade e da elegância.
